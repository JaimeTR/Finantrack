# **App Name**: TrackWise

## Core Features:

- Transaction Logging: Users can record income and expenses, specifying the amount, date, category, and description of each transaction. Includes type (income/expense) and amount in local currency.
- Financial Report Generation: Generates a financial report from transaction history, visualizing spending habits and trends over time.
- AI-Driven Budgeting Tool: Analyzes transaction history using AI as a tool to suggest personalized monthly budget recommendations, categorized by spending habits. Offers a 'total budget' configuration and sub-budgets per category.
- Saving Goals Tracker: Set and track progress toward financial goals, like paying off student loans. Track amount saved towards each goal.
- User Profile and Settings: Allows users to customize profile information, app settings, and set preferences such as dark mode.
- Firebase Authentication: Implements user authentication through Firebase Auth, with email/password login and the option to sign in with Google.
- Firebase Firestore Integration: Uses Firestore to store user transaction data, budget details, and saving goals.
